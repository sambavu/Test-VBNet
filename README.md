# VB.NET Console Application

Un projet d'application console VB.NET simple.

## Prérequis

- .NET 10.0 SDK ou supérieur

## Compilation et exécution

### Compiler le projet
```bash
dotnet build
```

### Exécuter l'application
```bash
dotnet run
```

## Structure du projet

- `Program.vb` - Point d'entrée de l'application
- `VBNetApp.vbproj` - Fichier de configuration du projet

## Tâches disponibles

Vous pouvez utiliser les commandes suivantes dans VS Code :
- **Ctrl+Shift+B** : Compiler le projet
- **F5** : Déboguer l'application
